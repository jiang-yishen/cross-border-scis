# 跨境海外仓供应链智能决策系统 (Cross-Border SCIS)

基于 Streamlit 构建的跨境电商海外仓供应链智能决策系统，覆盖从需求预测到库存调拨的全流程。

## 功能模块

- **需求预测分析**：Prophet + XGBoost 融合预测模型
- **补货计划看板**：智能补货建议 + 可视化看板
- **库存健康监控**：多维度库存健康度评估
- **调拨建议**：多仓间智能调拨策略
- **采购物流跟踪**：采购全流程追踪
- **使用指南**：系统操作指导文档
- **运维中心**：问题反馈 + 系统维护管理

## 技术栈

- Python 3.11
- Streamlit
- Prophet / XGBoost
- Plotly
- Pandas / NumPy

## 部署

本项目部署在 [Hugging Face Spaces](https://huggingface.co/spaces)。
